# 39-Language App Store Locale Plan

This app targets these 39 App Store metadata locales:

ar-SA, ca, cs, da, de-DE, el, en-AU, en-CA, en-GB, en-US, es-ES, es-MX, fi, fr-CA, fr-FR, he, hi, hr, hu, id, it, ja, ko, ms, nl-NL, no, pl, pt-BR, pt-PT, ro, ru, sk, sv, th, tr, uk, vi, zh-Hans, zh-Hant.

The first pass creates uploadable Fastlane metadata for every locale. The source product positioning is:

- A lightweight ERP tool for small Shaxian snack shops.
- Daily business report, material inventory, customer records, and cashflow ledger.
- Offline-first, simple enough for owner-operated shops.

Next localization steps:

1. Add `Localizable.xcstrings` for in-app UI text.
2. Generate localized screenshots per locale, not copied screenshots.
3. Review localized keywords before final App Store submission.
