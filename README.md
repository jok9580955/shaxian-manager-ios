# 沙县小吃店铺管家-经营日报-材料库存

面向沙县小吃、小吃档口和夫妻店的 iOS 经营工具。正式 App 名称为“沙县小吃店铺管家-经营日报-材料库存”，第一版主定位为“经营日报 + 材料库存”，同时保留高峰期出餐计时，方便店主每天看流水、管材料、记客户和成本。

## MVP 功能

- 经营日报：首页展示今日收入、材料成本、净流水。
- 材料库存：记录原料余量、供应商和补货提醒。
- 客户管理：沉淀回头客、团餐客户和联系方式。
- 财务记录：记录收入、进货、平台费用等流水。
- 出餐计时：蒸饺、拌面、炖罐三条出餐线并行倒计时。
- 39 语言上架准备：`fastlane/metadata` 生成 App Store 文案目录，`Localizable.xcstrings` 接入 App 内固定 UI 文案，`InfoPlist.xcstrings` 接入桌面显示名，语言清单见 `md/locales_39.md`。

## 视觉方向

图标用红橙暖色表达沙县小吃店的热气和烟火感，中间用日报单、库存格、金额符号表达“经营日报、材料库存、财务记录”，比单纯餐饮图标更贴近小企业主的 ERP 工具定位。
